## 关于本站

**Agent Craft** 是一个记录 Agent 开发实战与思考的技术博客。内容聚焦于如何基于 LLM 构建真正可用的 Agent——从模型调用、工具编排、任务规划、记忆系统，到生产级部署的工程实践。

这里的文章有两个特点：

- **实战导向**：每篇文章都伴随可复现的代码与踩坑经验，不空谈概念。
- **可交互**：关键篇章会嵌入可真实运行的 Agent Demo，读者可以在文章里直接点按钮观察 Agent 的执行轨迹（tool calls、中间思考、最终输出）。

## 关于作者

正在 AI Agent 方向持续深耕的开发者。本站既是学习笔记的沉淀，也希望能帮助同样在这条路上的人少走弯路。

## 技术栈

- 站点：[Astro](https://astro.build/) + [Tailwind CSS](https://tailwindcss.com/)，基于 [AstroPaper](https://github.com/satnaing/astro-paper) 主题二次开发
- 交互式 Demo：React Island 组件 + 后端 Agent 运行时
- Agent 框架：LangChain / LangGraph / Deep Agents
